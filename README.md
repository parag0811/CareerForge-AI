# 🚀 CareerForge AI

An AI-powered career guidance platform designed to help students discover the right career path, identify skill gaps, generate personalized learning roadmaps, and prepare for interviews.

---

## ✨ Features

### 🧠 AI Career Analysis
- Resume analysis simulation
- Skill assessment dashboard
- Career match recommendations
- Industry demand insights

### 🎯 Skill Gap Detection
- Identify missing skills
- Priority-based recommendations
- Career readiness evaluation

### 🗺 Personalized Learning Roadmaps
- 30-Day Plan
- 90-Day Plan
- 180-Day Plan
- Daily learning tasks

### 🎤 AI Mock Interviews
- HR Interview Simulation
- Technical Interview Simulation
- Performance Scoring
- Improvement Suggestions

### 📊 Student Dashboard
- Progress Tracking
- Skill Growth Monitoring
- Daily Goals
- Placement Readiness Metrics

### 🌐 Regional Language Support
- English
- Hindi
- Marathi
- Tamil
- Punjabi
- Bengali
- Telugu

---

## 🎯 Problem Statement

Many students struggle to understand:

- Which career path suits them
- Which skills they are missing
- How to prepare effectively
- How interview-ready they are

CareerForge AI addresses these challenges through an interactive AI-powered platform.

---

## 💡 Solution

CareerForge AI provides:

1. Career Analysis
2. Skill Gap Detection
3. Personalized Roadmaps
4. Mock Interviews
5. Progress Tracking

All within a single platform.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/careerforge-ai.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
